# 🧠 Nuvia

**Turns simple ideas into professionally optimized prompts automatically.**

---

## 🚀 What is Nuvia?

Nuvia is a 100% offline web application built in a single HTML file that transforms basic ideas or raw prompts into structured, professional-grade prompts optimized for AI tools.

It automatically detects the type of project and selects the most suitable prompt format to maximize output quality.

---

## ⚙️ Key Features

- 🧠 Automatic prompt optimization
- 📂 Smart project type detection
- 🧾 Multiple output formats:
  - Text (structured prompt)
  - XML
  - JSON
  - Markdown
- ⚡ 100% offline functionality (no backend, no APIs)
- 💾 Local history (temporary storage)
- 🎨 Modern UI inspired by Apple / Notion / SaaS tools
- 📱 Mobile-first responsive design
- ✨ Smooth AI-like animations and UX flow

---

## 🧩 How it works

1. User enters an idea or raw prompt
2. Nuvia analyzes the input locally
3. It detects the type of project (app, image, video, writing, etc.)
4. It selects the best prompt format automatically
5. It generates a structured, optimized prompt
6. User copies and uses it in any AI tool

---

## 🔌 Offline First

Nuvia works completely offline once loaded in the browser.

### No internet required for:
- Prompt generation
- Analysis
- Optimization
- History storage

### Internet only required for:
- Accessing external links (e.g. blog)
- Sending feedback via external form
